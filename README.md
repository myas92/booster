# Response Format 

It's sensitive to languages(`fa/en`)

**Note**: language in header of request

**Success**
```javascript
{
    "status": "success",
    "data": {
        "message": "something",
        "extraData": "something"
    }
}
```
**Error**
```javascript
// For language:fa
{
    "status": "error",
    "data": {
        "status_code": 200,
        "error_code": 1001,
        "timestamp": "2022-05-09T11:31:34.858Z",
        "path": "/api/v1/auth/register",
        "message": "شماره همراه تکراری است"
    }
}
```
